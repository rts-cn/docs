# 脚本语言的选择

## 简介

\[注意：此页面是从旧的MediaWiki复制过来的，迄今为止已经非常陈旧了。如同维基百科所形容的，它有很多问题，包括客观性问题。Lua解决了一些问题，Perl解决了其他问题，诸如此类。选择最适合当前工作的工具。\]

简而言之: [Lua](../Modules/mod_lua_1048944.mdx#--)。

更深入的回答: 使用适合你的工作的工具，但要注意其限制。

[JavaScript](JavaScript/index.mdx#api)非常流行，但并不意味着它是嵌入式电话编程的最佳选择。它比Lua更重且使用完全不同的FreeSWITCH接口方法。

[Python](../Modules/mod_python_1048940.mdx#eol)非常流行，但并不意味着它是嵌入式电话编程的最佳选择。它的嵌入性不如Lua。而且，大多数FreeSWITCH开发人员都是Perl的粉丝。:)

[Perl](../Modules/mod_perl_1048930.mdx#about)仍然流行，并且是嵌入式电话编程的一个不错的选择。然而，对于这种情况，Lua仍然比Perl更轻便，且不缺少Perl拥有的任何功能，包括正则表达式。

底线是：没有理由不使用Lua。
