## 介绍

本页面包含适用于基于XML的电话（如7945、7965、7970型号）的Cisco示例XML配置。

请使用自己的设置替换 `Freeswitch_IP_Address`、`Freeswitch_Extension`、`Freeswitch_Password` 和 `Freeswitch_Voicemail`。

此配置已在基于XML的Cisco电话上的SIP45.8-5-2S版本中经过测试。G.722宽带支持应无需额外设置，并在配置中包含了一个示例速拨条目。

基于SIP的Cisco IP电话将使用DHCP选项66来确定其呼叫控制服务器。基于Cisco SCCP的IP电话将使用DHCP选项150来确定其呼叫控制服务器。如果要将现有的Cisco Unified CallManager电话转换为SIP，或者它们是作为SCCP购买的，您可能会发现将这两个选项都编程到DHCP服务器中是很有价值的，用于自动化TFTP软件升级和配置。

若要重新启动IP电话，请按下“Settings”按钮，然后快速输入以下按键：\*\*#\*\*

若要解锁IP电话，请按下“Settings”按钮，然后输入\*\*#和输入您的电话密码。默认密码为Cisco，且区分大小写。自从发布了8.0系列版本后，Cisco已经锁定了绝大多数SIP设置，使其无法由用户修改 - 解锁您的电话对于进行恢复出厂设置或更改电话IP地址非常有用。

若要擦除电话配置，请按照上述解锁电话的步骤，然后按下“More”软键，最后点击“Erase”。

要获取有关通话的详细统计信息，请快速按两次蓝色的帮助按钮（在全局设置-即7945G上有一个蓝色问号图标）。
