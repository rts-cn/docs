## 什么是 RTS？

RTS 是一个实时解决方案，也是一个实时 SIP 服务器。包含两个部分：rtss（RTS Server）和 rts （RTS 客户端）。

- RTS 是一个 FreeSWITCH 衍生版。
- 有稳定的发行版。
- 开源。
- 最大程度兼容 FreeSWITCH。
- 有自己的 Bug 修复和扩展。
- 以及很多[新特性](/rts/features/)
- 和一些[扩展模块](/rts/modules/)
