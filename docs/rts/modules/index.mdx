# RTS Modules

RTS支持第三方模块。

## mod_rustit

[mod_rustit](https://github.com/rts-cn/mod_rustit)是一个用 Rust 语言写的模块。

## mod_shine

一个MP3录音模块。

[mod_shine](https://github.com/rts-cn/mod_shine)

## mod_vnc

与VNC交互的模块。

[mod_vnc](https://github.com/rts-cn/mod_vnc)

## mod_x11

与X11交互的模块。

[mod_x11](https://github.com/rts-cn/mod_x11)

## mod_aac

AAC编解码模块。

[mod_aac](https://github.com/rts-cn/mod_aac)

# mod_srs

SRS 对接模块。

[mod_srs](https://github.com/rts-cn/rts/tree/main/src/mod/endpoints/mod_srs)
