# FAQ

RTS 常见问题。

## 什么是 RTS？

RTS is a Derivative work of FreeSWITCH, Not a Fork。

RTS 是一个 FreeSWITCH 衍生品，不是一个 Fork。

RTS 旨在解决但不限于以下问题：

- 稳定的发行版
- 更实用的默认配置
- 更多的 Feature
- 国产化适配
- 更好的控制接口
- UI
- 还有好多[新特性](/rts/features/)
- 和一些[扩展模块](/rts/modules/)

RTS 期待：

- 更多的用户
- 更多的贡献者

## 为什么做 RTS？

我们为 FreeSWITCH 做了很多贡献，我们也会一如即往地做贡献。跟所有开源软件一样，开源软件其实鼓励做衍生品。
